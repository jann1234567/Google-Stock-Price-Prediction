# Google-Stock-Price-Prediction using LSTM and Recursive Method
This project uses an LSTM neural network to predict Google’s closing stock price from the previous three trading days. It also applies a recursive method, where each predicted value is reused as input to generate subsequent future predictions.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- TensorFlow
- Keras

## Dataset
The dataset contains daily Google stock-price information from January 2010 to December 2022.
The dataset was obtained from Kaggle: https://www.kaggle.com/datasets/alirezajavid1999/google-stock-2010-2023/data
